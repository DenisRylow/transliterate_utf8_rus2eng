Рылов Денис Сергеевич 2016.

Реализован алгоритм транслитерации текста с русского на английский в кодировке utf-8 согласно стандарту ISO 9 (ГОСТ 7.79).
Входной файл "input.txt", выходной файл "output.txt". Входной файл должен иметь BOM метку.

Программа копирует "input.txt" в "output.txt", производя замену русских букв на эквивалентные латинские буквосочетания.
Программа написана на С++11.
